# IRP

This repository contains the code to run the semi-automatic tool that can simulate lava flows in the folder LavaGenerationTool, this is run through pipeline.ipynb. DownloadingData has the code that was seperatly used to download Magellan radar images for training and testing the tool.

The deliverables folder contains the report and the folder ExampleResults contains examples the input and output image of the tool. The TrainingGANcode.ipynb notebook has the code that was used to train the model, the model wieghts after training is saved in the LavaGenerationTool folder so that it can be used whilst running the pipeline.
